Mock My Days

Productivity + accountability with a fun roast vibe.

Expo React Native Supabase PostgreSQL Java

Overview

Mock My Days is a productivity app built during a hackathon. It helps users plan tasks, stay consistent, and get motivational “mock/roast” feedback to keep momentum.

What I did

  • Designed the core flow: create tasks → schedule → progress tracking
  • Built the mobile UI in Expo + React Native
  • Connected backend using Supabase (PostgreSQL) for storing user/task data
